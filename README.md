# Employee Management System

## Project Overview

A full-stack Employee Management System built using FastAPI, SQLAlchemy, SQLite, HTML, CSS, and JavaScript.

## Features

* Add Employee
* View Employees
* Update Employee
* Delete Employee
* Search Employees by Department
* Dashboard Employee Count
* REST API Integration
* SQLite Database Storage

## Tech Stack

### Backend

* Python
* FastAPI
* SQLAlchemy
* SQLite

### Frontend

* HTML
* CSS
* JavaScript

## API Endpoints

### Employee APIs

* POST /employees
* GET /employees
* GET /employees/{id}
* PUT /employees/{id}
* DELETE /employees/{id}

### Search API

* GET /search?department=IT

## Project Architecture

Frontend (HTML + JavaScript)
↓
FastAPI Backend
↓
SQLAlchemy ORM
↓
SQLite Database
